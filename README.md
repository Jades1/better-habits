# Better Habits

A simple daily habit tracker. Open `index.html` in any browser — no install needed.

## Features
- Track daily habits with one-tap check-off
- Streak counters
- Stats and a 30-day completion chart
- Data saved locally in the browser (localStorage)

## Roadmap
- macOS notification nudges (reminders even when the app is closed)
- Per-habit charts
- Export/backup data
